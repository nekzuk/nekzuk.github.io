About NAI Smart Studio

NAI Smart Studio is a standalone Windows software that makes NovelAI, an AI image generation service, easier to use. It features various production support functions not found in the official web UI, realizing a comfortable image generation workflow.

Basic Usage

  1. Obtain a NovelAI account
  2. Get an API key from the NovelAI official website
  3. Enter the API key in NAI Smart Studio settings
  4. Input prompts and start generation

For detailed instructions, please refer to the online manual.

Key Features

NovelAI Core Functions

  • Access to almost all basic image generation features
  • Text-to-Image and Image-to-Image
  • NovelAI Potion support and Precise Reference functionality

Production Support

  • User-friendly and polished UI
  • Automatic batch generation
  • Wildcard support
  • Proprietary upscaling feature
  • Prompt search and history management
  • Tag completion and preset features
  • Eagle Linkage features

Privacy Protection

  • All data stored locally
  • Online connection only for NovelAI API communication
  • Prompts and images are not sent externally

System Requirements

  • OS: Windows 10/11 (64bit)
  • Memory: 8GB or more recommended
  • Storage (Free space): Min 800MB / Max 8GB
  • Internet connection: Required (NovelAI API communication only)
  • NovelAI subscription: Required (Tablet/Scroll/Opus)
