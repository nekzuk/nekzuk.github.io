Overview

AI Paint Cleaner is a comprehensive image editing tool designed for efficiently editing multiple images. It combines powerful AI inpainting capabilities with numerous batch processing features and integration with AutoMosaic, making it an all-in-one solution for illustration editing.

  • Offline processing: Primarily local processing (some API features require network connection)
  • Batch processing: Process many images at once
  • Preset formats: JPG / PNG / BMP / GIF / WEBP

Supported Languages

  • Japanese
  • English
  • Chinese (Simplified/Traditional)
  • Korean

Key Features

Designed primarily for AI-generated image editing:

  • Sequential display of images within a folder
  • AI-powered object removal and inpainting
  • Automatic object selection
  • Standard editing tools (pen, eraser, fill, etc.)
  • Mosaic tool
  • Selection & bookmark tool
  • Image metadata & embedded info display
  • Format conversion - single or batch (JPG, PNG, WEBP, PDF)
  • Resize / AI upscaling / background removal - single or batch
  • Tag generation (Tagger) - single or batch
  • Color correction (AI & manual) - single or batch
  • Metadata removal - single or batch

How to Use

  1. Select an image (open file or drag & drop)
  2. Edit images by switching through images in the folder
  3. Apply batch processing to images in the folder as needed

System Requirements

  • OS: Windows 10/11 (64-bit)
  • Memory: 8GB+ recommended
  • Storage: 15GB or more
  • GPU: NVIDIA GPU with CUDA support / VRAM: 8GB+ recommended
  • It can run without a GPU, but some AI processing will be limited.
