Overview 概要

AI Paint Cleaner is a comprehensive image editing tool designed for efficiently editing multiple images. It combines powerful AI inpainting capabilities with numerous batch processing features and integration with AutoMosaic, making it an all-in-one solution for illustration editing.

複数画像を効率的に編集＆調整がおこなえるツールです。AIによる強力なインペイント機能と数多くのバッチ処理、 自動モザイクツール(AutoMosaic)の連携機能が内蔵され、イラスト修正のオールインワンツールとなっています。

  • Offline processing: Primarily local processing (some API features require network connection)
  • Batch processing: Process many images at once
  • Preset formats: JPG / PNG / BMP / GIF / WEBP
  • オフライン動作: 基本的にローカル処理（一部API利用機能はネットワーク接続が必要）
  • バッチ処理: 複数画像をまとめて処理
  • 対応画像形式: JPG / PNG / BMP / GIF / WEBP

How to Use 基本的な使い方

  3. Apply batch processing to images in the folder as needed
