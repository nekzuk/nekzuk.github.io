Overview

Automatically detects specific regions in images and batch-processes multiple images. Offers the highest detection accuracy in its price range (free to a few thousand yen). Operates offline by default for privacy protection. Try the trial version!

  • Offline: no uploads, no cloud processing
  • Batch: process many images at once
  • Presets: pick what to detect and the output style
  • Supported formats: JPG / PNG / BMP / GIF / WEBP
  • Processing speed: 1-3 sec (GPU) / 5-8 sec (CPU)

How to use

  1. Select a preset
  2. Select an input folder
  3. Adjust settings (e.g., mosaic size)
  4. Click Run to start processing

System requirements

  • OS: Windows 10/11 (64-bit)
  • Memory: 8GB+ recommended
  • Storage (available/free space): 8GB or more
  • GPU: NVIDIA GPU / VRAM: 4GB+ recommended
  • It can run without a GPU, but processing will be slower.
