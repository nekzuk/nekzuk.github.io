Overview

AutoMosaic Video is a Windows desktop application that automatically detects and tracks targets in video, then applies the selected effect. Video is processed locally on your PC and is not uploaded for processing.

  • Detection and tracking: follows targets across frames
  • Offline processing: no uploads or cloud processing
  • Fast GPU acceleration: faster inference on NVIDIA GPUs
  • GUI / CLI: graphical and command-line workflows
  • Preflight: reports blocking issues and conversion warnings before processing

Detectable targets / Outputs

Detectable targets

  • People
  • Faces (general, female, and male) and eyes
  • Standard text and all text
  • License plates (EU, Japan, and general)
  • Twelve bundled presets for different targets

Outputs

  • Mosaic
  • Blur
  • Solid fill with a selectable color

Post-edit Materials (Full Version)

  • Video containing only processed regions
  • Grayscale mask video

How to use

  1. Add videos with Add Files or Add Folder
  2. Select a Detection Preset and Processing Mode
  3. Select Mosaic, Blur, or Solid Fill and adjust its settings
  4. Confirm the output container, codec, quality, and destination
  5. Start processing and review any preflight warnings
  6. Review the complete output after processing

System Requirements

  • OS: Windows 10/11 (64-bit)
  • Memory: 8GB+ recommended
  • Storage (available/free space): 8GB or more
  • GPU: NVIDIA GPU recommended
  • CPU-only processing is supported but may be significantly slower.

Supported Formats & Video Notes

Input

Main Output

Please Note

  • Some videos may differ from the original in motion or color
  • Additional audio, subtitles, and similar content are not preserved
  • A confirmation message appears before processing long videos
