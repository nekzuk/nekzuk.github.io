概要

AutoMosaic Videoは、動画内の対象を自動検出・追跡して画像処理を適用するWindows向けデスクトップアプリです。動画はPC上で処理し、処理素材をクラウドへアップロードしません。

  • 自動検出・追跡: 動画内の対象を継続して追跡
  • オフライン処理: 動画のアップロード不要、クラウド処理なし
  • 高速GPU処理: NVIDIA GPUによる高速処理に対応
  • GUI / CLI: 画面操作とコマンドラインの両方に対応
  • preflight: 開始できない理由や変換上の注意を処理前に表示

検出対象 / 出力

検出対象

  • 人物
  • 顔（全体・女性・男性）と目
  • 通常テキスト・全テキスト
  • ナンバープレート（EU・日本・汎用）
  • 用途別の同梱プリセット12種類

出力

  • モザイク
  • ぼかし
  • 塗りつぶし（色指定）

後編集用素材（製品版）

  • 処理部分のみの動画
  • 白黒マスク動画

基本的な使い方

  1. 「ファイル追加」または「フォルダ追加」で入力動画を選択
  2. 検出設定と処理モードを選択
  3. モザイク・ぼかし・塗りつぶしと関連設定を調整
  4. 主出力の形式・codec・品質と出力先を確認
  5. 処理を開始し、preflightの警告内容を確認
  6. 完了後に出力動画全体を確認

動作環境

  • OS: Windows 10/11(64bit)
  • メモリ: 8GB以上推奨
  • ストレージ（空き容量）: 8GB以上
  • GPU: NVIDIA GPU推奨
  • CPUのみでも動作しますが、処理時間は大幅に長くなる場合があります。

対応形式・動画処理上の注意

入力

MP4 / MOV / AVI / MKV / WebM / M4V / WMV / TS / MTS / M2TS

主出力

  • MP4 / MKV
  • H.264 / H.265
  • 容量優先 / 標準 / 高品質
  • 先頭音声ストリームの保持に対応

ご確認ください

  • 動画によっては、動きや色合いが元動画と異なる場合があります
  • 追加の音声や字幕などは保持されません
  • 長い動画では、処理前に確認メッセージを表示します

更新確認にインターネット接続を使用します。
