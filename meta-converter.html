Overview

Meta Converter is specifically designed for AI-generated images, allowing you to convert and compress images while maintaining all generation metadata. The converted images can be directly imported into WebUI's PNGinfo, and it also supports Eagle software library conversion.

  • Batch conversion: Convert multiple images while preserving metadata
  • Format support: PNG/WEBP → JPEG/PNG/WEBP conversion
  • File size reduction: PNG → JPEG reduces file size to 1/10 with significant compression
  • WebUI compatible: Direct import to WebUI after conversion
  • Quality adjustment: Fine-tune quality and image resizing options
  • Eagle support: Eagle library conversion support
  • CLI support: Command-line interface support

How to use

  1. Select a folder containing images or drag and drop image files/folders
  2. Adjust conversion format, quality settings, etc.
  3. Click "Start Conversion" button

System requirements

  • OS: Windows 10/11 (64-bit)
  • Memory: 8 GB or more recommended
  • Storage: 30MB+ free space require

No GPU required. Runs on CPU for universal compatibility.
